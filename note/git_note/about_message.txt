命令行的书写
	版本回退：
		$git log
			$git log--pretty=oneline
				清晰书写
			查看日志
			存在id便于制定控制位置
			
		$git reset --hard id
		$git push -f
			强制push到远程
		$git pull
			在push之前确定是最新版
		$git reflog
			即使出现了错误reset也可以重新更改
	
	
	message修改：
		$git commit --amend
			修改message
				可以在曾经的message基础上进行增删查改
		$git rebase -i HEAD5
			其中，HEAD~5表示最近的5个，后面的5可以改成其他数字。
